Temperature and Humidity Monitoring

This tool allows you to visualize temperature and humidity data collected with a DHT22 sensor. The data is collected every second using a Parallax Activity Board and Simple IDE software. Since this is an offline monitoring system, it records data from a starting point (0) and continues without a real-time clock reference.
