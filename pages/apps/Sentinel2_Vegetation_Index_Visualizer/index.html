Interactive Vegetation Index Computing

The Sentinel-2 Vegetation Index Visualizer is an interactive Google Earth Engine (GEE) tool that lets users draw a polygon, select a date, and instantly compute dozens of vegetation indices from Sentinel-2 imagery. The app automatically applies cloud masking, overlays indices on RGB basemaps, and displays a dynamic color legend for interpretation.

Developed at the Digital Agriculture Laboratory, UC Davis, this tool supports vegetation monitoring, crop health assessment, and environmental research applications.

Technical Information

Data Source

  • Sentinel-2 MSI Level-1C imagery
  • Automatic cloud masking
  • 10-20 meter spatial resolution
  • Google Earth Engine cloud computing platform

Available Vegetation Indices

  • NDVI - Normalized Difference Vegetation Index
  • EVI - Enhanced Vegetation Index
  • SAVI - Soil-Adjusted Vegetation Index
  • NDCI - Normalized Difference Chlorophyll Index
  • ARI - Anthocyanin Reflectance Index
  • GNDVI - Green NDVI
  • MCARI - Modified Chlorophyll Absorption Ratio
  • MSI - Moisture Stress Index
  • NDMI - Normalized Difference Moisture Index
  • NDWI - Normalized Difference Water Index
  • PSSRB1 - Pigment-Specific Simple Ratio
  • SIPI1 - Structure-Insensitive Pigment Index
  • PSRI - Plant Senescence Reflectance Index
  • CHL_REDEDGE - Chlorophyll Red-Edge Index
  • mARI - Modified Anthocyanin Reflectance Index
  • NDMI STRESS - NDMI for Crop Moisture Stress
